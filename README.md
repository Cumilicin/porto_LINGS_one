# Portofolio Alingga

Website portofolio satu halaman untuk link bio Instagram, LinkedIn, dan kontak profesional. Tampilan dibuat mobile-first, modern, dan tetap cocok dilihat perusahaan.

## Cara membuka

Buka salah satu URL ini di browser:

- `http://localhost/porto/` jika Apache XAMPP sedang menyala.
- `http://localhost/porto/index.php` jika ingin membuka file PHP secara langsung.

## Yang perlu diganti

Edit `index.php`, lalu ubah data di bagian paling atas file:

- Nama, headline, dan deskripsi singkat
- Link WhatsApp, Instagram, email, dan GitHub
- Daftar project di bagian `$works`
- Daftar layanan di bagian `$services`

Bagian karya ada di section `Karya pilihan`. Ganti judul, deskripsi, kategori, hasil, dan gambar sesuai project asli.

## File utama

- `index.php` untuk data, struktur, dan konten.
- `styles.css` untuk tampilan responsive.
- `assets/profil-crop.jpg` untuk foto profil.
- `assets/Screenshot 2026-05-15 160504.png` untuk preview project.
